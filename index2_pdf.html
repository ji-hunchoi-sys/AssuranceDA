사용 가이드

            

                

                    
자금일보 은행대사 검증이란?

                    
자금팀에서 작성한 자금일보(일일 자금 보고서)의 내용이 실제 은행 입출금 내역과 정확히 일치하는지 교차검증하는 프로세스입니다.

                    

                    
검증 항목

                    
    
                        
  • 거래 매칭: 은행 내역의 각 거래가 자금일보에 정확히 반영되었는지
    • 
                        
  • 금액 일치: 입금/출금 금액이 원 단위까지 정확한지
    • 
                        
  • 누락 검출: 은행에는 있는데 자금일보에 빠진 거래
    • 
                        
  • 초과 검출: 자금일보에는 있는데 은행 내역에 없는 거래
    • 
                        
  • 잔액 검증: 일별 마감 잔액이 은행 잔액과 일치하는지
    • 
                        
  • 날짜 검증: 거래일자가 정확히 기재되었는지
    • 
                        
  • 주요 거래 누락 검증: 은행 거래 중 상대적으로 금액이 큰 주요 건이 자금일보에 모두 반영되었는지 별도 확인
    • 
                    

                    

                    
사용 방법

                    
    
                        
  1. 자금일보 파일(PDF 또는 Excel)을 왼쪽에 업로드
    2. 
                        
  2. 은행에서 다운로드한 입출금 내역(Excel/CSV/PDF)을 오른쪽에 업로드
    3. 
                        
  3. 은행 파일의 열 매핑을 확인 (자동 감지됨)
    4. 
                        
  4. "대사 검증 시작" 버튼 클릭
    5. 
                        
  5. 결과 확인 및 불일치 항목 조치
    6. 
                    

                    

                    
지원 파일 형식

                    
자금일보: PDF, Excel(.xlsx/.xls), CSV, 이미지(PNG, JPG, BMP, TIFF)

                    
은행 거래내역: Excel(.xlsx/.xls), CSV, PDF, 이미지(PNG, JPG, BMP, TIFF)

                    
대부분의 국내 은행 거래내역 다운로드 파일을 지원합니다 (KB국민, 신한, 우리, 하나, IBK기업, NH농협, SC제일 등).
인터넷뱅킹에서 PDF로 다운로드한 거래내역 명세서도 자동으로 파싱합니다.

                    
이미지 파일: Tesseract.js OCR을 사용하여 이미지에서 텍스트를 자동 인식합니다. 가능한 선명하고 기울어지지 않은 캡쳐/스캔 이미지를 사용하세요. 첫 인식 시 한국어 학습 데이터를 다운로드하므로 약간의 시간이 소요될 수 있습니다.

                    

                    
주요 입출금 기능 (v2.3 신규)

                    
자금일보 생성 시: 은행 거래 중 상대적으로 금액이 큰 건(상위 20% 등)을 자동으로 "주요 입출금"으로 표시합니다. 실무에서 담당자가 별도로 주기 표시하던 작업을 자동화합니다.

                    
대사 검증 시: 은행 거래내역의 주요 대금액 건이 자금일보에 모두 반영되었는지 별도로 체크합니다. 실무자가 놓칠 수 있는 중요 거래 누락을 사전에 감지합니다.

                    
주요 거래 기준은 설정 페이지에서 "상위 10%/20%/30%"로 조정할 수 있습니다.
